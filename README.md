## Project Overview

This Power BI solution integrates data from 15+ hospital sources including patient records, doctor schedules, medical tests, surgeries, medicine stock, billing transactions, room availability, and suppliers.

The dashboard helps hospital management:

- Identify revenue leakage  
- Understand patient flow and seasonal trends  
- Optimize doctor workload and appointment scheduling  
- Improve diagnostic and pharmacy performance  
- Reduce stockouts and inventory cost  
- Increase room/Premium occupancy  
- Enhance overall operational efficiency
